Description: <p>Optolith is a project I started 2016. It is a character creation and management software for the pen-and-paper game *The Dark Eye* from Ulisses Spiele (*Das Schwarze Auge* in German). If you know pen-and-paper games, you probably know you usually need to do a lot of calculations. Optolith solves that issue. Its primary target are beginners, who want to be guided so they don't need to keep track of all the dependencies and costs.</p><p>I'm still the main contributor to Optolith, although there are a few other contributors due to the project being available on GitHub as Open Source. I develop the core and design and implement UI and UX concepts and improvements.</p>
